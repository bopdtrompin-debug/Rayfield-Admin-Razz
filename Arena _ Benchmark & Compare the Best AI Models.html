    Saya telah mengemaskini kod laman web supaya sistem logik penjanaan key tersebut lebih mesra dan mudah dipadankan tepat (1:1) dengan bahasa pengaturcaraan Lua yang digunakan oleh Roblox.

    Seterusnya, ini ialah Skrip Roblox Admin (menggunakan Rayfield UI) yang direka khas untuk menggunakan dan mengesahkan key yang dijana dari laman web tadi. Skrip ini akan mengira sendiri key semasa berdasarkan masa komputer pengguna untuk memastikan ia sepadan dengan key di laman web anda.

    Anda boleh salin dan gunakan skrip di bawah pada mana-mana executor Roblox anda:

    Lua
    -- Memuatkan Rayfield UI Library
local Rayfield = loadstring(game:HttpGet('https://sirius.menu/rayfield'))()

-- Fungsi untuk mendapatkan Window Index semasa (mengikut zon masa 30 minit)
local function getWindowIndex()
    -- os.time() adalah masa dalam saat. 30 minit = 1800 saat.
    return math.floor(os.time() / 1800)
end

-- Fungsi untuk menjana key berdasarkan algoritma yang SAMA seperti laman web anda
local function getValidKey()
    local windowIndex = getWindowIndex()
    
    local num1 = windowIndex * 1337
    local num2 = windowIndex * 7331
    
    -- Format menjadi 4 abjad Hex (Huruf Besar)
    local str1 = string.format("%04X", num1 % 65536)
    local str2 = string.format("%04X", num2 % 65536)
    
    return string.format("NEXUS-%s-%s", str1, str2)
end

-- Menyemak key rahsia yang sepatutnya untuk 30 minit ini
local currentKey = getValidKey()

-- Membina Tetingkap Utama (Window) Rayfield
local Window = Rayfield:CreateWindow({
    Name = "Nexus Admin Hub",
    LoadingTitle = "Nexus Admin System",
    LoadingSubtitle = "Protected by Nexus Key",
    ConfigurationSaving = {
        Enabled = true,
        FolderName = "NexusHub",
        FileName = "NexusConfig"
    },
    Discord = {
        Enabled = false,
        Invite = "link_discord_anda",
        RememberJoins = true 
    },
    KeySystem = true, -- Mengaktifkan Key System
    KeySettings = {
        Title = "Nexus Hub | Key System",
        Subtitle = "Pengesahan diperlukan",
        Note = "Dapatkan key di laman web rasmi kami. Key bertukar setiap 30 minit!",
        FileName = "NexusKey", 
        SaveKey = false, -- Jangan simpan key supaya mereka perlu masukkan key baru bila luput
        GrabKeyFromSite = false,
        Key = {currentKey} -- Menggunakan array kerana Rayfield support multiple keys
    }
})

-- Membina Tab Pertama: "Admin"
local AdminTab = Window:CreateTab("Admin Commands", 4483362458) 

local Section = AdminTab:CreateSection("Tetapan Pemain (Player Settings)")

AdminTab:CreateSlider({
    Name = "WalkSpeed (Kelajuan)",
    Range = {16, 200},
    Increment = 1,
    Suffix = "Speed",
    CurrentValue = 16,
    Flag = "WalkSpeedSlider",
    Callback = function(Value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value
    end,
})

AdminTab:CreateSlider({
    Name = "JumpPower (Kuasa Lompatan)",
    Range = {50, 500},
    Increment = 1,
    Suffix = "Power",
    CurrentValue = 50,
    Flag = "JumpPowerSlider",
    Callback = function(Value)
        game.Players.LocalPlayer.Character.Humanoid.JumpPower = Value
    end,
})

AdminTab:CreateButton({
    Name = "Infinite Jump (Lompat Tanpa Had)",
    Callback = function()
        local UserInputService = game:GetService("UserInputService")
        local Player = game.Players.LocalPlayer
        
        UserInputService.JumpRequest:Connect(function()
            if Player.Character and Player.Character:FindFirstChild("Humanoid") then
                Player.Character.Humanoid:ChangeState(Enum.HumanoidStateType.Jumping)
            end
        end)
        Rayfield:Notify({
            Title = "Berjaya",
            Content = "Infinite Jump telah diaktifkan!",
            Duration = 3,
            Image = 4483362458,
        })
    end,
})

local Section2 = AdminTab:CreateSection("Skrip Berguna")

    end,

-- Load konfigurasi dan siapkan Hub untuk mula

    Bagaimana Ia Berfungsi:

    1. Skrip Lua mengambil masa tepat Unix Time (sistem masa seluruh dunia yang sentiasa selaras).
    2. Ia membahagikan masa tersebut kepada window 30 minit (math.floor(os.time() / 1800)).
    3. Kemudian, ia mendarab, menukar kepada format Hex (16) lalu memadankan string "NEXUS-XXXX-XXXX" sama persis seperti yang dihasilkan oleh logik laman web tadi.
    4. Apabila pengguna menekan butang Generate pada web anda, mereka akan menerima key ini. Apabila mereka meletakkan salinan (paste) pada Roblox, UI Rayfield akan menerima log masuk masuk tersebut!
